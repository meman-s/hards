# Алгоритмы на графах

Граф задаётся списком смежности `graph: List[List[int]]`, `n` — число вершин, вершины от `0` до `n-1`.

| Файл | Алгоритм |
|------|----------|
| `01_bfs_undirected.py` | Обход неориентированного графа в ширину (BFS) |
| `02_dfs_undirected.py` | Обход неориентированного графа в глубину (DFS), рекурсия и стек |
| `03_bfs_directed.py` | Обход ориентированного графа в ширину (BFS) |
| `04_dfs_directed.py` | Обход ориентированного графа в глубину (DFS) |
| `05_topological_sort.py` | Топологическая сортировка (DFS с цветами и алгоритм Кана) |

В каждом файле в `if __name__ == "__main__"` приведён пример использования.
